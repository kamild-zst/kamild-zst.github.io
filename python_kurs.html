Zmienne i typy danych

Zmienna to nazwane miejsce w pamięci komputera, w którym przechowujesz dane. W Pythonie nie musisz deklarować typu — interpreter sam go wykryje.

💡 Zasada Python jest dynamicznie typowany — typ zmiennej zależy od wartości, którą do niej przypisujesz.

Podstawowe typy danych

Python 3 
# --- Liczby całkowite (int) ---

rok         = 2024
temperatura = -7

# --- Liczby zmiennoprzecinkowe (float) ---
pi     = 3.14159
cena   = 49.99
wynik  = 1_000_000.5   # _ jako separator (czytelność)

# --- Łańcuchy znaków (str) ---

pozdrowienie = 'Cześć, świecie!'
wielolinia   = """To jest
tekst w wielu
liniach."""

# --- Wartości logiczne (bool) ---
jest_aktywny = True
jest_admin   = False

# --- Brak wartości (NoneType) ---
wynik_bledu  = None

# Sprawdzenie typu:
print(type(wiek))          # <class 'int'>
print(type(pi))           # <class 'float'>
print(type(imie))         # <class 'str'>
print(type(jest_aktywny)) # <class 'bool'>

Pętle for i while

Pętle pozwalają powtarzać bloki kodu — bez pisania tego samego dziesiątki razy. for iteruje po sekwencjach, while działa, dopóki warunek jest spełniony.

Pętla for — range()

Przykład 1 — range() 
# range(stop) — od 0 do stop-1
for i in range(5):
    print(f"i = {i}")

print("---")

# range(start, stop) — od start do stop-1
for i in range(1, 6):
    print(f"  liczba: {i}")

print("---")

# range(start, stop, step) — co krok
for i in range(0, 20, 4):
    print(f"  krok 4: {i}")

print("---")

# Odliczanie wstecz
for i in range(5, 0, -1):
    print(f"  start: {i}")

Zwracanie wielu wartości

Przykład 5 — multi return 
def statystyki(lista):
    minimum  = min(lista)
    maksimum = max(lista)
    srednia  = sum(lista) / len(lista)
    return minimum, maksimum, round(srednia, 2)

dane = [12, 45, 7, 89, 33, 21]
mn, mx, sr = statystyki(dane)   # rozpakowanie krotki

print(f"Min: {mn}, Max: {mx}, Śr: {sr}")

Listy (tablice)

Lista to uporządkowana kolekcja elementów. Może zawierać dowolne typy danych, można ją modyfikować, a elementy są dostępne przez indeks (od 0).

Tworzenie i dostęp do elementów

Indeksowanie i slicing 
kolory = ["czerwony", "zielony", "niebieski", "żółty", "fioletowy"]

print(kolory[0])    # czerwony  (pierwszy)
print(kolory[2])    # niebieski (trzeci)
print(kolory[-1])   # fioletowy (ostatni)
print(kolory[-2])   # żółty (przedostatni)

# Slicing [start:stop:krok]
print(kolory[1:4])    # [zielony, niebieski, żółty]
print(kolory[:3])     # [czerwony, zielony, niebieski]
print(kolory[2:])     # [niebieski, żółty, fioletowy]
print(kolory[::-1])   # odwrócona lista

Geometria z math 
import math

def pole_kola(r):
    return round(math.pi * r ** 2, 4)

def obwod_kola(r):
    return round(2 * math.pi * r, 4)

def przeciwprostokatna(a, b):
    return round(math.sqrt(a**2 + b**2), 4)

# Wyniki
print(f"Koło r=5:  pole={pole_kola(5)}, obwód={obwod_kola(5)}")
print(f"Trójkąt 3-4-?: c = {przeciwprostokatna(3, 4)}")

# Kąty stopień → radian
for stopnie in [0, 30, 45, 60, 90]:
    rad = math.radians(stopnie)
    print(f"  sin({stopnie:2}°) = {math.sin(rad):.4f}")
